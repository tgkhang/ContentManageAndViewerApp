## Development

1. Clone the repository
2. Set up each project following their individual README instructions
3. Run each project in development mode

## Technology Stack

- **Backend**: NestJS
- **Frontend**: React + TypeScript + Vite
- **Database**: MongoDB, S3 (AWS)
